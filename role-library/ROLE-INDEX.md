## 🎭 角色扮演使用方法

### 1. 选择角色

告诉大圣爸爸你想扮演的角色，例如：
- "扮演 Frontend Developer"
- "我需要 UX Designer 的建议"
- "用 Product Manager 的视角分析"

### 2. 加载角色设定

我会：
1. 读取对应角色的 `.md` 文件
2. 理解角色职责和技能
3. 按照角色设定与你互动

### 3. 角色扮演

我会：
- 使用角色的专业术语
- 从角色视角思考问题
- 提供角色专长的建议

**大圣爸爸：** "用 Product Manager 的视角分析这个功能"

```
从 PM 角度，我会考虑：

2. 商业价值
   - 对业务指标的影响
   - ROI 分析
   
3. 优先级
   - RICE 评分
   - 与 OKR 的对齐
...
```
