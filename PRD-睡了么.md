## 1. 产品概述

### 1.1 产品愿景

让每一个失眠的夜晚不再孤单，用科学方法重建生物钟，用社交力量养成好睡眠。

### 1.2 产品目标

- **短期目标（6个月）：** 帮助用户建立规律的睡眠习惯，7天打卡留存率达到40%
- **中期目标（1年）：** 成为国内领先的睡眠健康管理App，月活用户突破50万
- **长期目标（3年）：** 构建睡眠健康生态圈，拓展至冥想、压力管理等相关领域

### 1.3 产品定位

**睡了么**是一款面向失眠人群和作息不规律人群的移动端应用，通过**生物钟调节算法** + **社交监督机制**双引擎驱动，帮助用户建立健康的睡眠习惯。

**核心价值主张：**
- 🧠 科学：基于睡眠医学的生物钟调节方案
- 👥 陪伴：找到同样在深夜奋斗的小伙伴
- 🏆 激励：游戏化成就系统让坚持变得有趣

### 1.4 目标市场

中国约有3亿失眠人群，其中18-35岁年轻人占比超过60%。主要痛点包括：
- 工作压力大导致入睡困难
- 手机依赖导致熬夜成瘾
- 缺乏科学的睡眠指导
- 独自坚持难以形成习惯

- 连续打卡徽章（7天/21天/100天）
- 早睡早起勋章

## 5. 功能详细设计

### 5.1 睡眠打卡功能

**输入：**
- 用户点击"准备睡觉"或"我醒了"按钮
- 系统自动获取当前时间戳
- 可选：用户手动调整时间（补打卡场景）

**输出：**
- 睡眠记录存入数据库
- 更新个人打卡日历
- 触发小组状态更新
- 推送次日早晨起床提醒

**逻辑流程：**
```
1. 用户点击打卡按钮
2. 校验时间合理性（入睡时间应在18:00-10:00之间）
3. 记录时间戳，计算睡眠时长（如果是起床打卡）
4. 更新连续打卡计数
5. 检查是否达成成就条件
6. 同步到小组成员可见状态
7. 返回打卡成功反馈
```

**异常处理：**
- 重复打卡：提示"今日已打卡"
- 时间异常：如凌晨3点起床打卡，提示确认
- 网络失败：本地缓存，网络恢复后同步

### 5.2 生物钟调节功能

**输入：**
- 用户当前平均入睡/起床时间（基于历史打卡数据）
- 用户目标入睡/起床时间
- 用户期望的调节周期（7天/14天/21天）

**输出：**
- 每日个性化睡眠计划
- 每日提醒时间动态调整
- 调节进度可视化

**逻辑流程：**
```
1. 计算时间差：目标时间 - 当前时间
2. 根据调节周期计算每日调整量（建议15-30分钟/天）
3. 生成每日计划表（入睡时间、起床时间、提醒时间）
4. 每日0点更新当天计划
5. 用户完成打卡后更新进度
6. 如连续3天未完成，主动询问是否需要调整计划
```

**算法说明：**
- 调节速度 = 时间差 / 调节周期
- 如调节速度 > 30分钟/天，建议延长周期
- 周末可适当放宽，允许1小时弹性

### 5.3 社交监督功能

**输入：**
- 小组创建信息（名称、人数上限、公开/私密）
- 邀请码或分享链接
- 打卡状态变更事件

**输出：**
- 小组首页状态展示
- 组员提醒通知
- 小组聊天消息

**逻辑流程：**
```
1. 创建小组：生成唯一小组ID和邀请码
2. 加入小组：验证邀请码，检查人数上限
3. 打卡同步：用户打卡后，更新小组状态缓存
4. 提醒功能：选择未打卡组员，发送系统通知
5. 聊天功能：消息存入小组聊天历史
```

**权限设计：**
- 群主：可解散小组、移除成员、修改小组信息
- 普通成员：可邀请他人、发送提醒、参与聊天

### 5.4 成就系统功能

**输入：**
- 用户打卡记录
- 成就规则配置表

**输出：**
- 成就解锁通知
- 徽章展示
- 排行榜数据

**成就规则示例：**
| 成就名称 | 达成条件 | 徽章图标 |
|----------|----------|----------|
| 初出茅庐 | 连续打卡7天 | 🌱 |
| 坚持不懈 | 连续打卡21天 | 🔥 |
| 百日勇士 | 连续打卡100天 | 👑 |
| 早睡达人 | 连续7天23:00前入睡 | 🌙 |
| 早起鸟儿 | 连续7天7:00前起床 | 🐦 |
| 社交达人 | 邀请5位好友加入 | 🤝 |

**逻辑流程：**
```
1. 每次打卡后检查成就条件
2. 如达成新成就，写入用户成就表
3. 推送成就解锁通知
4. 更新排行榜数据
```

| 小睡眠 | 白噪音质量好 | 功能单一、缺乏打卡机制 | 完整打卡体系 + 成就激励 |
| AutoSleep | 自动追踪、数据详细 | 仅支持Apple Watch、价格高 | 免费、跨平台、社交属性 |

### B. 技术选型说明

- **跨平台框架：** React Native - 一套代码同时支持iOS和Android，降低开发成本
- **数据库：** MongoDB - 灵活的数据结构适合快速迭代，支持海量用户数据
- **推送服务：** 极光推送 - 国内覆盖率高，到达率稳定
- **数据分析：** 神策数据 - 支持用户行为分析，帮助产品优化
