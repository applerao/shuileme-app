## 一、内容审核标准

### 1.1 审核原则

**核心价值观：** 温暖、真实、正向、安全

| 原则 | 说明 | 执行标准 |
|------|------|---------|
| 温暖 | 内容应传递情感温度，避免冷漠攻击 | 鼓励共情表达，限制冷嘲热讽 |
| 真实 | 鼓励真实情感表达，反对虚假编造 | 允许艺术加工，禁止恶意虚构 |
| 正向 | 倡导积极心态，但不强制正能量 | 允许负面情绪表达，引导正向转化 |
| 安全 | 确保内容合法合规，保护用户隐私 | 严格遵守法律法规，零容忍违规 |

### 1.2 内容分级体系

| 等级 | 标识 | 定义 | 处理方式 |
|------|------|------|---------|
| L1 | 🟢 优质 | 真实温暖、有深度、有共鸣 | 推荐曝光、加精、作者奖励 |
| L2 | 🔵 正常 | 符合规范、无明显问题 | 正常展示 |
| L3 | 🟡 待优化 | 质量一般、需改进 | 限流、引导优化 |
| L4 | 🟠 敏感 | 擦边内容、需人工复核 | 人工审核、暂不公开 |
| L5 | 🔴 违规 | 违反规范、有害内容 | 删除、处罚作者 |

### 1.3 禁止内容清单（红线）

**绝对禁止（一经发现立即删除并处罚）：**

| 类别 | 具体内容 | 处罚措施 |
|------|---------|---------|
| 政治敏感 | 攻击国家政策、领导人、政治制度 | 永久封禁 + 上报 |
| 违法违规 | 黄赌毒、暴力恐怖、违禁品交易 | 永久封禁 + 上报 |
| 人身攻击 | 辱骂、诽谤、歧视特定群体 | 封禁 7-30 天 |
| 隐私泄露 | 曝光他人隐私信息（电话、地址等） | 删除 + 封禁 3-7 天 |
| 虚假信息 | 谣言、诈骗、虚假医疗/投资建议 | 删除 + 封禁 7-30 天 |
| 广告营销 | 未经授权的商业推广、引流 | 删除 + 警告/限流 |
| 自残自杀 | 宣扬、教唆自残自杀行为 | 删除 + 封禁 + 心理干预 |
| 版权侵犯 | 盗用他人作品、未授权转载 | 删除 + 警告 |

### 1.4 限制内容清单（黄线）

**限制传播（限流处理，需优化后恢复正常）：**

| 类别 | 具体内容 | 处理方式 |
|------|---------|---------|
| 过度负面情绪 | 纯粹抱怨、无建设性的消极内容 | 限流 50%，引导正向表达 |
| 低质内容 | 无意义灌水、重复发布、内容过短 | 限流 30%，提示优化 |
| 敏感话题 | 涉及两性、宗教等需谨慎讨论 | 人工复核，适度限流 |
| 争议内容 | 可能引发激烈争论的话题 | 关闭评论或预审评论 |
| 营销擦边 | 隐晦推广、暗示性引流 | 删除 + 警告 |

### 1.5 审核流程

```
用户发布内容
    ↓
机器初审（AI 模型）
    ↓
├─ 明确合规 → 自动通过（L1/L2）
├─ 疑似违规 → 人工复审（L3/L4）
└─ 明确违规 → 自动拦截（L5）
    ↓
人工复审（9:00-22:00，30 分钟内）
    ↓
├─ 通过 → 发布
├─ 需修改 → 通知用户优化
└─ 不通过 → 删除 + 处罚
    ↓
用户申诉（可选）
    ↓
二审（24h 内）→ 最终裁决
```

### 1.6 审核时效要求

| 审核类型 | 时效要求 | 责任人 |
|---------|---------|--------|
| 机器初审 | 实时（<5 秒） | 系统自动 |
| 人工复审 | 30 分钟内（9:00-22:00） | 审核专员 |
| 用户申诉 | 24 小时内 | 审核主管 |
| 举报处理 | 2 小时内 | 审核专员 |

## 二、优质内容标准

### 2.1 优质内容定义

**优质内容 = 真实情感 × 表达质量 × 共鸣价值**

### 2.2 优质内容评分维度

| 维度 | 权重 | 评分标准（1-5 分） |
|------|------|-------------------|
| 真实性 | 30% | 5=高度真实感人，3=一般真实，1=疑似虚构 |
| 情感深度 | 25% | 5=深刻洞察，3=有情感表达，1=表面化 |
| 表达质量 | 20% | 5=文笔优美流畅，3=通顺，1=混乱 |
| 共鸣价值 | 15% | 5=引发广泛共鸣，3=部分共鸣，1=无共鸣 |
| 正向引导 | 10% | 5=积极正向，3=中性，1=消极负面 |

**优质内容阈值：** 综合得分 ≥ 4.0 分

### 2.3 优质内容类型示例

#### 🌟 深度自我探索类
> "30 岁这一年，我终于学会和自己和解。不再用别人的标准衡量自己，不再为'应该'而活。日记里的每一页，都是我与自己对话的痕迹。"

**特点：** 真实、有深度、引发思考

#### 💖 情感共鸣类
> "今天收到陌生人的留言：'你的日记治愈了我'。原来我的文字，真的可以温暖另一个人。这大概就是坚持记录的意义。"

**特点：** 温暖、有共情、传递善意

#### 🎯 成长记录类
> "Day 30 | 情绪管理挑战完成。从易怒到平和，日记帮我看见了情绪背后的需求。分享 3 个对我有用的方法..."

**特点：** 有方法、可复制、正向引导

#### 📖 故事叙述类
> "那个雨夜，我写下了第一封'给未来的信'。三年后的今天，我重新打开它，泪流满面。"

**特点：** 有情节、有转折、引人入胜

### 2.4 优质内容激励

| 激励类型 | 获取条件 | 奖励内容 |
|---------|---------|---------|
| 精选推荐 | 得分≥4.5，审核通过 | 首页推荐位 +500 心语币 + 精选勋章 |
| 周榜前十 | 周点赞数前 10 | 周榜勋章 +1000 心语币 + 流量扶持 |
| 月度之星 | 月综合评分前 5 | 月度之星勋章 + 实体周边 + 专访机会 |
| 优质创作者 | 连续 4 周产出优质内容 | 优质创作者认证 + VIP 月卡 + 专属标识 |

## 三、内容推荐算法

### 3.1 推荐系统架构

```
用户行为数据 ──┐
              ├──→ 特征工程 ──→ 召回层 ──→ 排序层 ──→ 重排层 ──→ 推荐结果
内容特征数据 ──┘
```

### 3.2 召回策略

| 召回源 | 权重 | 说明 |
|--------|------|------|
| 协同过滤 | 30% | 相似用户喜欢的内容 |
| 内容相似度 | 25% | 与用户历史阅读内容相似 |
| 热门内容 | 20% | 全站热门、 trending 内容 |
| 地理位置 | 10% | 同城用户内容 |
| 新人冷启动 | 15% | 优质内容随机曝光 |

### 3.3 排序因子

**排序公式（简化版）：**
```
Score = w1×CTR + w2×互动率 + w3×完读率 + w4×时效性 + w5×质量分 + w6×多样性惩罚
```

| 因子 | 权重 | 计算方式 |
|------|------|---------|
| CTR（点击率） | 25% | 点击数/曝光数 |
| 互动率 | 25% | (点赞 + 评论 + 分享)/阅读数 |
| 完读率 | 20% | 完整阅读数/阅读数 |
| 时效性 | 15% | 根据发布时间衰减 |
| 质量分 | 10% | 内容审核评分 |
| 多样性惩罚 | 5% | 避免同类内容过度集中 |

### 3.4 个性化推荐逻辑

```python
# 推荐算法伪代码
def recommend_content(user_id, limit=20):
    # 1. 获取用户画像
    user_profile = get_user_profile(user_id)
    
    # 2. 多路召回
    candidate_pool = []
    candidate_pool += collaborative_filtering(user_id, k=50)
    candidate_pool += content_similarity(user_id, k=50)
    candidate_pool += hot_content(k=30)
    candidate_pool += location_based(user_id, k=20)
    
    # 3. 去重与过滤
    candidate_pool = deduplicate(candidate_pool)
    candidate_pool = filter_blocked(candidate_pool, user_id)
    
    # 4. 精排
    scored_items = []
    for item in candidate_pool:
        score = calculate_score(user_profile, item)
        scored_items.append((item, score))
    
    # 5. 重排（多样性、新鲜度）
    final_list = rerank(scored_items, diversity=0.3)
    
    return final_list[:limit]
```

## 五、违规处理流程

### 5.1 违规等级与处罚

| 违规等级 | 违规类型 | 处罚措施 | 申诉权限 |
|---------|---------|---------|---------|
| 轻度违规 | 低质灌水、轻微营销 | 警告 + 限流 3 天 | 可申诉 |
| 中度违规 | 擦边内容、重复违规 | 删除内容 + 封禁 3-7 天 | 可申诉 |
| 重度违规 | 人身攻击、隐私泄露 | 删除内容 + 封禁 15-30 天 | 可申诉 |
| 严重违规 | 黄赌毒、政治敏感 | 永久封禁 + 上报 | 不可申诉 |

### 5.2 违规处理流程

```
违规发现（系统/用户举报）
    ↓
内容下架（立即）
    ↓
违规判定（审核专员，30 分钟内）
    ↓
处罚执行（系统自动）
    ↓
通知用户（站内信 + Push）
    ↓
用户申诉（可选，7 天内）
    ↓
申诉复审（审核主管，24h 内）
    ↓
├─ 申诉成功 → 恢复内容 + 撤销处罚
└─ 申诉失败 → 维持处罚
```

### 5.3 用户举报机制

**举报入口：** 每篇日记右下角"..." → "举报"

**举报类型：**
- 违法违规
- 人身攻击
- 隐私泄露
- 虚假信息
- 广告营销
- 色情低俗
- 其他

**举报奖励：**
- 有效举报：+50 心语币/次
- 月度举报达人：额外奖励 500 心语币

**恶意举报处罚：**
- 恶意举报≥3 次：警告
- 恶意举报≥5 次：封禁举报权限 30 天

### 5.4 处罚通知模板

**轻度违规通知：**
> "您的内容因'[违规原因]'被限流处理。请优化内容质量，共同维护良好的社区环境。如有疑问可申诉。"

**中度违规通知：**
> "您的内容因'[违规原因]'已被删除，账号封禁 X 天。请遵守社区规范，详情查看《内容运营规范》。"

**重度违规通知：**
> "您的账号因严重违规已被封禁 X 天。如再次违规将永久封禁。请珍视账号，文明发言。"

**永久封禁通知：**
> "您的账号因严重违反法律法规已被永久封禁。根据相关规定，我们已上报有关部门。"

### 5.5 黑名单管理

| 黑名单类型 | 纳入条件 | 管理方式 |
|-----------|---------|---------|
| 设备黑名单 | 同一设备多次违规注册 | 设备 ID 封禁 |
| IP 黑名单 | 同一 IP 大量违规账号 | IP 限流/封禁 |
| 用户黑名单 | 严重违规用户 | 永久封禁 + 身份标记 |
| 关键词黑名单 | 高频违规关键词 | 自动拦截 |

## 六、内容运营数据监控

### 6.1 核心指标

| 指标类别 | 指标名称 | 计算公式 | 目标值 |
|---------|---------|---------|--------|
| 内容生产 | 日新增日记数 | 当日发布日记总数 | ≥10,000 |
| 内容质量 | 优质内容率 | 优质内容数/总内容数 | ≥15% |
| 内容消费 | 人均阅读数 | 总阅读数/DAU | ≥20 |
| 内容互动 | 互动率 | (点赞 + 评论 + 分享)/阅读数 | ≥8% |
| 内容安全 | 违规内容率 | 违规内容数/总内容数 | ≤1% |

### 6.2 数据看板

**日报指标：**
- 新增日记数、活跃创作者数
- 精选内容数、违规内容数
- 举报处理数、用户申诉数

**周报指标：**
- 内容质量趋势、互动率变化
- 创作者成长情况、激励发放统计
- 违规类型分布、处罚效果分析

### 6.3 预警机制

| 指标 | 预警阈值 | 响应措施 |
|------|---------|---------|
| 违规内容率 | >2% | 加强审核、调整策略 |
| 用户举报数 | 日增>50% | 排查热点事件、增加人手 |
| 优质内容率 | <10% | 优化激励、调整推荐 |
| 用户申诉率 | >5% | 复盘处罚标准、培训审核 |
