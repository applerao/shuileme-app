## 五、内容安全规范

### 5.1 内容安全原则

**基本原则：** 合法合规、真实健康、积极向上

| 原则 | 说明 | 执行标准 |
|------|------|---------|
| 合法性 | 符合国家法律法规 | 零容忍违法违规内容 |
| 真实性 | 鼓励真实表达 | 禁止恶意虚构、谣言 |
| 健康性 | 传递正向价值 | 限制过度负面情绪 |
| 安全性 | 保护用户权益 | 禁止侵犯隐私、人身攻击 |

### 5.2 内容安全红线

**绝对禁止内容（发现立即删除并上报）：**

| 类别 | 具体内容 | 处置措施 |
|------|---------|---------|
| 政治类 | 攻击国家政策、领导人、政治制度 | 删除 + 封禁 + 上报 |
| 违法类 | 黄赌毒、暴力恐怖、违禁品 | 删除 + 封禁 + 上报 |
| 危害类 | 自残自杀宣扬、教唆 | 删除 + 封禁 + 心理干预 |
| 侵权类 | 侵犯他人隐私、名誉、知识产权 | 删除 + 封禁 |
| 欺诈类 | 诈骗、虚假医疗/投资建议 | 删除 + 封禁 + 上报 |

### 5.3 内容审核机制

```
用户发布
    ↓
AI 初审（实时）
    ↓
├─ 明确合规 → 自动通过
├─ 疑似违规 → 人工复审
└─ 明确违规 → 自动拦截
    ↓
人工复审（30 分钟内）
    ↓
├─ 通过 → 发布
├─ 修改 → 通知用户
└─ 不通过 → 删除 + 处罚
    ↓
用户申诉（可选）
    ↓
二审（24h 内）→ 最终裁决
```
