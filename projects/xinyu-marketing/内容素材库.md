## 一、品牌故事（3-5 篇）

### 故事 1：《心语的诞生》

**标题：** 为什么我们做了一个让人"说心里话"的 App？

**正文：**

2023 年的一个深夜，我们的创始人林然坐在北京的出租屋里，打开朋友圈，写了一大段话，然后删掉。又打开微博，写了几句，还是删掉。

最后，他对着手机屏幕发呆，突然意识到：在这个拥有 10 亿网民的时代，竟然没有一个地方，可以让人安心地说出心里话。

朋友圈有熟人，微博有陌生人，但都没有"懂你的人"。

于是，心语诞生了。

心语不是一个社交软件，而是一个情绪避风港。在这里：
- 你可以匿名，不用担心被熟人认出
- 你可以倾诉，总有人愿意倾听
- 你可以被理解，因为这里的人都懂

我们不是要做一个"更大"的社交软件，而是要做一个"更温暖"的地方。

如果你也曾有过"想说却不敢说"的时刻，心语欢迎你。

### 故事 2：《第一个用户的故事》

**标题：** 她在心语说了第一句话，然后哭了

**正文：**

心语上线的第 3 天，我们收到了第一条用户反馈。

她叫小雅，26 岁，在上海工作。那天凌晨 2 点，她在应用商店下载了心语，发布了第一条树洞：

"今天被领导骂了，说我不够努力。可是我已经连续加班一个月了。好想回家，可是房租刚交了一年。"

5 分钟后，有 3 个人回复了她。有人说"抱抱你"，有人说"我懂你"，有人分享了自己的经历。

小雅给我们发邮件说："看到那些回复的时候，我突然哭了。不是因为委屈，是因为终于有人懂我。"

这就是心语存在的意义。

### 故事 3：《从 0 到 1 的坚持》

**标题：** 做一款"慢"产品，在这个快时代

**正文：**

在创业圈，"快"是王道。快速迭代、快速获客、快速变现。

但心语选择"慢"。

我们花 3 个月打磨匿名机制，确保用户隐私绝对安全；
我们花 2 个月训练匹配算法，让每个人都能找到懂自己的人；
我们花 1 个月审核每一条内容，保证社区温暖纯净。

有人问：这样不会错过风口吗？

我们的回答是：如果风口意味着牺牲用户体验，那我们宁愿错过。

心语要做的，不是一款"快消品"，而是一个可以陪伴用户很久的地方。

慢一点，没关系。只要方向是对的。

## 三、产品教程（5-10 篇）

### 教程 1：《3 分钟上手心语》

**步骤：**

1. **下载注册**
   - 应用商店搜索"心语"
   - 手机号注册（可选匿名昵称）
   
2. **发布第一条树洞**
   - 点击底部"+"按钮
   - 写下你的心事
   - 选择是否匿名
   
3. **找到懂你的人**
   - 系统会自动匹配相似用户
   - 也可以主动浏览树洞广场
   
4. **开始对话**
   - 点击匹配到的用户
   - 开始私密对话
   - 随时可以结束

**Tips：**
- 首次发布建议写 50 字以上，更容易获得回应
- 完善个人标签，匹配更精准
- 每天签到，获得积分奖励

---

### 教程 2：《如何保护你的隐私》

**隐私设置指南：**

1. **匿名设置**
   - 进入"我的" → "隐私设置"
   - 开启"完全匿名"模式
   
2. **可见范围**
   - 可选择"仅匹配用户可见"
   - 或"所有人可见"
   
3. **屏蔽功能**
   - 遇到不适用户，一键屏蔽
   - 屏蔽后对方无法再联系你
   
4. **数据删除**
   - 可随时删除自己的所有内容
   - 注销账号后数据永久清除

## 四、情感文章（20-30 篇）

### 情感文章 1

**标题：** 成年人的崩溃，都是静悄悄的

**正文：**

你有没有过这样的时刻？

明明很难过，却在别人问"你怎么了"的时候，说"没事"；
明明想哭，却在厕所里擦干眼泪，笑着走出去；
明明撑不住了，却还在朋友圈发"今天也是元气满满的一天"。

成年人的世界，连崩溃都要选时间、选地点。

不能在公司，不能在家里，不能在熟人面前。

只能在深夜的朋友圈，写一段又删掉；
只能在无人的地铁上，让眼泪偷偷流下来；
只能在心语这样的地方，说一句"我真的很累"。

如果你也经历过这样的时刻，请记住：
- 你不是矫情，你只是太累了
- 你不是脆弱，你只是需要休息
- 你不是一个人，有很多人和你一样

允许自己崩溃，也是一种勇气。

---

### 情感文章 2

**标题：** 为什么我们越来越不敢说心里话？

**正文：**

小时候，我们有委屈会跟妈妈说；
上学时，我们有烦恼会跟朋友说；
工作后，我们却谁都不敢说了。

为什么？

因为朋友圈有同事，微博有熟人，连抖音都认识半个朋友圈。

说开心，怕被说炫耀；
说难过，怕被说矫情；
说迷茫，怕被说负能量。

于是，我们学会了"懂事"，学会了"成熟"，学会了把情绪调成静音模式。

但情绪不会消失，只会累积。

累积到一定程度，就会爆发。

所以，找一个可以说心里话的地方，不是软弱，是自救。

心语，就是这样一个地方。

## 六、内容生产 SOP

### 6.1 内容创作流程

```
选题确定 → 素材收集 → 初稿撰写 → 审核修改 → 排版发布 → 数据复盘
```

### 6.2 内容质量标准

| 维度 | 标准 |
|------|------|
| 真实性 | 基于真实用户故事或场景 |
| 共鸣度 | 能引发目标用户情感共鸣 |
| 传播性 | 有分享价值，能引发讨论 |
| 品牌调性 | 温暖、治愈、不评判 |

### 6.3 内容审核清单

- [ ] 内容是否符合品牌调性？
- [ ] 是否有虚假夸大？
- [ ] 是否涉及敏感话题？
- [ ] 用户故事是否获得授权？
- [ ] 是否有错别字/语病？
- [ ] 配图是否合规？
