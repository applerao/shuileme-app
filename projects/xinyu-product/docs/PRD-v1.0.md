## 一、产品概述

### 1.1 产品愿景

**让每个人都能被理解，让每段关系都从真诚开始。**

心语是一款融合"日记记录"与"深度社交"的创新应用，通过性格测试和日记内容匹配，帮助用户找到真正懂自己的人。

### 1.2 产品定位

| 维度 | 描述 |
|------|------|
| **产品形态** | 日记工具 + 社交平台的融合体 |
| **目标用户** | 22-35 岁，追求深度关系、有表达欲的都市青年 |
| **核心价值** | 以日记为载体的低压力深度社交 |
| **差异化** | 先记录后匹配、基于文字的深度了解、渐进式信任 |
| **Slogan** | "遇见懂你的人，记录真实的自己" |

### 1.3 产品目标

#### 短期目标（0-6 个月）
- 完成 MVP 开发并上线
- 获取 10,000 名种子用户
- 日活达到 3,000+
- 用户留存率（7 日）达到 40%+

#### 中期目标（6-12 个月）
- 用户量达到 100,000+
- 日活达到 30,000+
- 付费转化率达到 8%+
- 建立核心用户社区

#### 长期目标（1-2 年）
- 用户量达到 1,000,000+
- 成为"深度社交"品类代表
- 探索多元化商业模式
- 建立品牌壁垒

## 五、功能详细说明

### 5.1 性格测试模块

#### 5.1.1 测试流程

```
开始测试 → 答题（30-40 题） → 进度展示 → 结果生成 → 结果解读 → 分享/保存
```

#### 5.1.2 题目设计

- **题型：** 情境选择题（二选一或四选一）
- **维度：** 外向/内向、感觉/直觉、思考/情感、判断/知觉
- **时长：** 5-8 分钟完成
- **交互：** 进度条、可暂停、自动保存

#### 5.1.3 结果展示

- **性格类型：** 16 种类型之一（如 INFJ、ENFP 等）
- **详细解读：** 性格特点、优势、盲点、适合的职业/关系
- **匹配建议：** 哪些性格类型与你更匹配
- **可视化：** 四维雷达图

#### 5.1.4 技术需求

- 测试题目可配置（后台管理）
- 结果算法可调整
- 支持 A/B 测试不同题目版本

---

### 5.2 日记功能模块

#### 5.2.1 写日记流程

```
点击写日记 → 选择模板（可选） → 选择心情 → 编辑内容 → 设置权限 → 发布
```

#### 5.2.2 日记模板

| 模板名称 | 使用场景 | 引导问题 |
|----------|----------|----------|
| 晨间日记 | 早上规划 | 今天最重要的 3 件事？期待什么？ |
| 晚间复盘 | 晚上总结 | 今天完成了什么？有什么遗憾？ |
| 情绪记录 | 情绪波动时 | 现在什么心情？发生了什么？ |
| 感恩日记 | 培养积极心态 | 今天值得感恩的 3 件事？ |
| 自由书写 | 随意记录 | 无引导，自由发挥 |

#### 5.2.3 权限设置

| 权限级别 | 可见范围 | 适用场景 |
|----------|----------|----------|
| 仅自己 | 只有作者可见 | 私密记录 |
| 部分公开 | 匹配用户可见 | 寻求共鸣 |
| 完全公开 | 所有人可见 | 分享观点 |

#### 5.2.4 技术需求

- 支持富文本编辑
- 图片压缩和 CDN 存储
- 权限实时生效
- 内容审核（AI+ 人工）

### 5.3 匹配系统模块

#### 5.3.1 匹配算法

**匹配维度：**
1. **性格匹配（40% 权重）**
   - 性格类型兼容性
   - 核心价值观相似度

2. **日记内容匹配（35% 权重）**
   - 话题相似度
   - 情绪共鸣度
   - 表达风格匹配

3. **兴趣匹配（15% 权重）**
   - 标签重合度
   - 社区参与重叠

4. **基础条件匹配（10% 权重）**
   - 年龄、城市、婚恋状态等

#### 5.3.2 推荐策略

- **每日推荐：** 5-10 人，早上 8 点更新
- **刷新机制：** 每日自动刷新，不可手动刷新（防止过度筛选）
- **去重机制：** 已推荐用户 30 天内不重复
- **冷启动：** 新用户优先推荐活跃用户

#### 5.3.3 匹配交互

- **喜欢：** 点击"心动"表达喜欢
- **跳过：** 点击"下一个"跳过
- **详情：** 查看对方资料和日记片段
- **匹配通知：** 双向喜欢后通知双方

#### 5.4.1 聊天功能

| 功能 | 描述 | 限制 |
|------|------|------|
| 文字消息 | 发送文字 | 单条 2000 字 |
| 图片消息 | 发送图片 | 最多 9 张/条 |
| 语音消息 | 发送语音 | 最长 60 秒 |
| 表情 | 发送表情 | 内置 + 自定义 |
| 已读回执 | 显示已读状态 | 可关闭 |
| 撤回 | 撤回消息 | 2 分钟内 |

#### 5.4.2 聊天安全

- **敏感词过滤：** 自动过滤违规内容
- **举报入口：** 聊天界面一键举报
- **拉黑功能：** 拉黑后无法接收对方消息
- **风险提示：** 检测诈骗话术并提示

#### 5.4.3 破冰辅助

- **开场建议：** 基于对方日记推荐开场白
- **话题推荐：** 聊天中推荐话题
- **共同点提示：** 显示双方共同兴趣

#### 5.5.1 隐私控制

#### 5.5.2 安全机制

- **举报处理：** 24 小时内处理

- 隐私政策明确告知

- 账号可注销

| 指标 | 定义 | 计算公式 | 目标值 |
