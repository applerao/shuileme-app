# 心语 App 项目

**项目定位：** 基于性格匹配的深度社交 + 日记记录应用

**Slogan：** 遇见懂你的人，记录真实的自己

---

## 项目结构

```

│   ├── 竞品分析报告.md      # 竞品分析
```
