## 一、品牌定位

### 1.1 品牌核心

**心语**是一款基于 AI 性格匹配的匿名日记社交应用，帮助用户找到真正理解自己的"日记友"。

**品牌使命：** 让每个孤独的灵魂都能找到共鸣

**品牌愿景：** 成为中国最大的深度情感交流平台

### 1.2 目标用户

| 用户画像 | 特征描述 |
|---------|---------|
| **核心用户** | 25-35 岁都市白领，情感细腻，渴望深度交流 |
| **次要用户** | 18-24 岁大学生，探索自我，寻求认同感 |
| **边缘用户** | 35-45 岁成熟人群，需要情感出口 |

### 1.3 品牌价值

- **温暖：** 像深夜的一杯热茶，治愈每个孤独时刻
- **治愈：** 在书写与倾听中找到内心的平静
- **深度：** 超越表面的社交，建立真正的情感连接

## 三、Logo 设计方案

### 3.1 方案一：对话气泡 + 心形

**设计理念：** 将对话气泡与心形巧妙融合，象征"用心交流"

**视觉元素：**
- 外层：柔和的对话气泡轮廓
- 内层：抽象的心形图案
- 色彩：渐变暖橙色

**适用场景：** App 图标、启动页、品牌主视觉

**AI 生成提示词：**
```
minimalist logo design, speech bubble merged with heart shape, 
warm orange gradient, soft rounded corners, clean vector style, 
white background, app icon, modern and friendly, 2D flat design
```

### 3.2 方案二：信笺 + 星光

**设计理念：** 日记如信件，匹配如星光，寓意"在茫茫人海中找到懂你的人"

**视觉元素：**
- 主体：折叠的信笺轮廓
- 点缀：三颗小星星环绕
- 色彩：暖黄色 + 淡紫色

**适用场景：** 品牌周边、会员体系、特殊活动

**AI 生成提示词：**
```
elegant logo design, folded letter envelope with three small stars, 
warm yellow and soft purple color scheme, minimalist line art, 
white background, sophisticated and dreamy, vector illustration
```

### 3.3 方案三：抽象声波 + 心跳

**设计理念：** 声音的振动与心跳的节奏，象征"情感的共鸣"

**视觉元素：**
- 主体：抽象的声波曲线
- 融合：心跳线元素
- 色彩：珊瑚粉渐变

**适用场景：** 动态 Logo、视频片头、动效设计

**AI 生成提示词：**
```
abstract logo design, sound wave merged with heartbeat line, 
coral pink gradient, flowing curves, dynamic and organic, 
white background, modern tech meets emotion, vector style
```

### 3.4 Logo 使用规范

| 场景 | 推荐方案 | 尺寸要求 |
|------|---------|---------|
| App 图标 | 方案一 | 1024×1024 px |
| 启动页 | 方案一 | 根据屏幕适配 |
| 网站 Favicon | 方案一简化版 | 32×32 px |
| 印刷物料 | 方案二 | 矢量格式 |
| 视频动效 | 方案三 | 动态 SVG |

## 五、品牌字体

### 5.1 中文字体

**主要字体：** 思源黑体 (Source Han Sans CN)

| 字重 | 使用场景 |
|------|---------|
| Light (300) | 大标题、展示文字 |
| Normal (400) | 正文、说明文字 |
| Medium (500) | 小标题、强调文字 |
| Bold (700) | 重要按钮、核心数据 |

**备选字体：** 苹方 (PingFang SC)、微软雅黑

### 5.2 英文字体

**主要字体：** Inter

| 字重 | 使用场景 |
|------|---------|
| Light (300) | 大标题 |
| Regular (400) | 正文 |
| SemiBold (600) | 按钮、强调 |
| Bold (700) | 重要数据 |

### 5.3 字体层级规范

```
H1 主标题：32px / Line-height 1.3 / Light
H2 副标题：24px / Line-height 1.4 / Medium
H3 小标题：18px / Line-height 1.5 / Medium
Body 正文：16px / Line-height 1.7 / Normal
Caption 说明：14px / Line-height 1.6 / Normal
Small 辅助：12px / Line-height 1.5 / Normal
```

## 六、品牌语调

### 6.1 核心调性

**温暖 · 治愈 · 深度**

### 6.2 语言风格

| 场景 | 语调 | 示例 |
|------|------|------|
| **欢迎语** | 温暖亲切 | "欢迎来到心语，这里有人愿意听你说心里话" |
| **引导文案** | 耐心细致 | "慢慢来，不用着急，写下此刻的心情就好" |
| **匹配成功** | 惊喜期待 | "你们有 87% 的相似度，也许 TA 真的懂你" |
| **错误提示** | 温和安抚 | "网络有点小情绪，稍后再试一次好吗？" |
| **会员推广** | 真诚推荐 | "如果心语陪伴了你很多夜晚，考虑支持我们继续前行" |

### 6.3 禁用表达

❌ "你必须..." → ✅ "建议你可以..."
❌ "操作失败" → ✅ "遇到了点小问题"
❌ "立即注册" → ✅ "开始你的心语之旅"
❌ "付费会员" → ✅ "心语守护者"

## 七、品牌故事

### 7.1 起源故事

在快节奏的都市生活中，我们都有太多话想说，却找不到合适的人倾听。

朋友圈太公开，微博太嘈杂，陌生人太遥远。我们需要的，是一个可以安心倾诉的空间，一个真正懂自己的人。

**心语**，由此诞生。

### 7.2 品牌叙事

> "深夜 11 点，小雅打开日记，写下今天的工作委屈。
> 
> 3 分钟后，系统匹配到了远在成都的阿宁——一个同样刚结束加班的设计师。
> 
> 阿宁回复：'我懂，上周我也遇到了类似的情况...'
> 
> 那一刻，小雅知道，自己不再孤单。"

### 7.3 品牌承诺

- **匿名安全：** 你的身份，只有你自己能决定何时公开
- **深度匹配：** 基于性格、价值观、兴趣的 AI 算法，找到真正的共鸣
- **温暖陪伴：** 每一次倾诉，都会有人认真倾听

### 7.4 品牌口号

**主口号：** 心语，让心事有归处

**备选口号：**
- 写下心情，遇见懂你的人
- 每一段心事，都值得被倾听
- 在心语，你从不孤单
