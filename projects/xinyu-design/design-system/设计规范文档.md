## 一、栅格系统

### 1.1 基础栅格

**移动端标准 (iOS/Android)**

```
┌────────────────────────────────────────┐
│            安全区域顶部                 │
├────────────────────────────────────────┤
│  16px  │  内容区域  │  16px            │
│  边距  │            │  边距             │
│        │            │                  │
│        │            │                  │
│        │            │                  │
│        │            │                  │
│        │            │                  │
├────────────────────────────────────────┤
│            安全区域底部                 │
└────────────────────────────────────────┘

屏幕宽度：375px (设计基准)
左右边距：16px
内容宽度：343px
```

### 1.2 栅格参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 列数 | 4 列 | 基础栅格列数 |
| 槽宽 | 16px | 列间距 (Gutter) |
| 边距 | 16px | 左右边距 (Margin) |
| 列宽 | 自适应 | 根据屏幕计算 |

### 1.3 响应式断点

| 设备类型 | 屏幕宽度 | 边距 | 列数 |
|---------|---------|------|------|
| 小屏手机 | < 375px | 12px | 4 列 |
| 标准手机 | 375px | 16px | 4 列 |
| 大屏手机 | > 414px | 20px | 4 列 |
| 平板 | > 768px | 24px | 8 列 |
| 桌面 | > 1024px | 32px | 12 列 |

### 1.4 布局示例

```
【通栏布局】
├─ 16px ──────────────────────────── 16px ─┤
├────────── 内容区域 (343px) ──────────────┤

【双列布局】
├─ 16px ─┬──── 16px ────┬──── 16px ────┬─ 16px ─┤
│        │   内容 A     │   内容 B     │        │
│        │   (148px)    │   (148px)    │        │

【三列布局】
├─ 16px ─┬── 12px ──┬── 12px ──┬── 12px ──┬─ 16px ─┤
│        │  内容 A   │  内容 B   │  内容 C   │        │
│        │  (95px)   │  (95px)   │  (95px)   │        │
```

## 二、间距规范

### 2.1 间距基准

**基础单位：** 4px

**间距序列：** 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96

### 2.2 常用间距

| 间距值 | 使用场景 |
|--------|---------|
| 4px | 图标与文字间距、紧凑元素 |
| 8px | 相关元素间距、标签间距 |
| 12px | 卡片内边距、列表项间距 |
| 16px | 标准内边距、模块间距 |
| 20px | 次级模块间距 |
| 24px | 页面内边距、大模块间距 |
| 32px | 分区间距、大卡片边距 |
| 40px | 大分区间距 |
| 48px | 页面级间距 |
| 64px+ | 特殊场景、留白区域 |

### 2.3 组件间距规范

#### 按钮内部间距
```
按钮高度：48px
左右内边距：24px
图标与文字间距：8px
```

#### 卡片内部间距
```
卡片边距：16px
内容块间距：12px
标题与内容间距：8px
```

#### 列表项间距
```
列表项高度：64px (标准)
左右内边距：16px
上下内边距：12px
列表项之间：0px (用分割线) 或 8px (卡片式)
```

#### 表单间距
```
表单项间距：16px
标签与输入框：8px
输入框与提示文字：4px
错误信息与输入框：4px
```

### 2.4 页面级间距

```
【标准页面结构】
├─ 24px ─┤ 顶部导航栏
├─ 16px ─┤ 导航栏与内容间距
├─ 内容区域
├─ 24px ─┤ 底部安全区域
```

## 三、颜色规范

### 3.1 色彩变量命名

**Design Token 命名规范：**

```
--color-{语义}-{强度}

示例：
--color-primary-base      (心语橙 #FF8A5B)
--color-primary-light     (浅橙 #FFB395)
--color-primary-dark      (深橙 #E66A3E)
--color-neutral-bg        (米白 #F9F7F5)
--color-text-primary      (深灰 #4A4643)
--color-text-secondary    (中灰 #9B9590)
```

### 3.2 完整色彩表

#### 主色系
| Token | 色值 | 用途 |
|-------|------|------|
| `--color-primary-base` | #FF8A5B | 主按钮、核心交互 |
| `--color-primary-light` | #FFB395 | 悬停、次要按钮 |
| `--color-primary-dark` | #E66A3E | 按压、强调文字 |

#### 辅助色系
| Token | 色值 | 用途 |
|-------|------|------|
| `--color-secondary-base` | #9B7EBD | 会员、特殊标识 |
| `--color-secondary-light` | #C4B5D9 | 紫色背景 |
| `--color-secondary-dark` | #7A5C99 | 紫色文字 |

#### 中性色系
| Token | 色值 | 用途 |
|-------|------|------|
| `--color-neutral-white` | #FFFFFF | 主背景、卡片 |
| `--color-neutral-bg` | #F9F7F5 | 次级背景 |
| `--color-neutral-border` | #E8E6E3 | 边框、分割线 |
| `--color-neutral-text-2` | #9B9590 | 次要文字 |
| `--color-neutral-text-1` | #4A4643 | 主要文字 |
| `--color-neutral-black` | #1A1817 | 重要强调 |

#### 功能色系
| Token | 色值 | 用途 |
|-------|------|------|
| `--color-success` | #7ED6A3 | 成功、完成 |
| `--color-error` | #FF6B8A | 错误、删除 |
| `--color-warning` | #FFC96B | 警告、提醒 |
| `--color-info` | #7EB8DD | 信息、链接 |

### 3.3 透明度使用规范

| 透明度 | 使用场景 |
|--------|---------|
| 100% | 实色背景、主要文字 |
| 80% | 次要文字、禁用状态 |
| 60% | 占位符、辅助信息 |
| 40% | 边框、分割线 |
| 20% | 背景装饰、遮罩底层 |
| 10% | 极淡背景、hover 反馈 |

### 3.4 深色模式色彩映射

```
浅色模式                    深色模式
─────────────────────────────────────────
#FFFFFF (白)        →       #1E1C1B (深灰)
#F9F7F5 (米白)      →       #12100F (深黑)
#4A4643 (深灰)      →       #E8E6E3 (浅灰)
#9B9590 (中灰)      →       #9B9590 (保持不变)
#FF8A5B (主橙)      →       #FF8A5B (保持不变)
```

## 七、动效规范

### 7.1 缓动曲线

```css
/* 标准缓动 */
--ease-standard: cubic-bezier(0.25, 0.1, 0.25, 1);

/* 进入缓动 */
--ease-in: cubic-bezier(0.55, 0, 0.55, 1);

/* 退出缓动 */
--ease-out: cubic-bezier(0.25, 0, 0.45, 1);

/* 弹性缓动 */
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 7.2 动效时长

| 类型 | 时长 | 使用场景 |
|------|------|---------|
| 快速 | 100-150ms | 小图标、状态变化 |
| 标准 | 200-250ms | 按钮、卡片、列表项 |
| 慢速 | 300-400ms | 页面切换、模态框 |
| 极慢 | 500ms+ | 引导页、特殊转场 |

### 7.3 标准动效库

#### 淡入淡出 (Fade)
```css
.fade-in {
  opacity: 0;
  animation: fadeIn 0.3s var(--ease-standard);
}

@keyframes fadeIn {
  to { opacity: 1; }
}
```

#### 滑入 (Slide In)
```css
.slide-in-right {
  transform: translateX(100%);
  animation: slideInRight 0.3s var(--ease-standard);
}

@keyframes slideInRight {
  to { transform: translateX(0); }
}
```

#### 缩放 (Scale)
```css
.scale-up {
  transform: scale(0.9);
  animation: scaleUp 0.2s var(--ease-bounce);
}

@keyframes scaleUp {
  to { transform: scale(1); }
}
```

#### 弹跳 (Bounce)
```css
.bounce {
  animation: bounce 0.4s var(--ease-bounce);
}

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
}
```

### 7.4 页面转场

| 转场类型 | 方向 | 时长 | 缓动 |
|---------|------|------|------|
| 推入 | 从右向左 | 300ms | standard |
| 弹出 | 从左向右 | 300ms | standard |
| 淡入 | 无方向 | 250ms | standard |
| 缩放 | 中心放大 | 250ms | bounce |
| 底部弹出 | 从下向上 | 300ms | standard |

## 八、无障碍规范

### 8.1 色彩对比度

| 内容类型 | 最小对比度 | 检查工具 |
|---------|-----------|---------|
| 正常文字 | 4.5:1 | WebAIM |
| 大文字 (>18px) | 3:1 | WebAIM |
| UI 组件 | 3:1 | WebAIM |
| 装饰元素 | 无要求 | - |

### 8.2 点击区域

```
【最小点击区域】
尺寸：44×44pt (iOS) / 48×48dp (Android)

【图标按钮】
图标：24×24px
点击区域：44×44px (含透明边距)
```

### 8.3 文字可读性

```
【最小字号】
正文：不低于 14px
辅助文字：不低于 12px

【最大行宽】
中文：不超过 35 个字符
英文：不超过 75 个字符
```

### 8.4 辅助功能

- 所有图标需有 `aria-label` 或 `alt` 文本
- 表单输入需有 `label` 关联
- 错误信息需与输入框关联
- 动态内容变化需有屏幕阅读器通知
