## 一、情绪板 (Moodboard)

### 1.1 视觉关键词

```
温暖 · 治愈 · 留白 · 手绘 · 柔和 · 深度 · 安静 · 陪伴
```

### 1.2 情绪参考

**色彩情绪：**
- 黄昏时分的暖橙色天空
- 深夜台灯下的柔和光晕
- 清晨薄雾中的淡紫色远山
- 热茶升起的温暖蒸汽

**质感情绪：**
- 棉麻布料的柔软触感
- 手绘插画的温度感
- 纸张的细腻纹理
- 水彩的晕染效果

**场景情绪：**
- 深夜书房，一人一灯一本日记
- 咖啡馆角落，安静书写
- 窗前远眺，思绪飘远
- 被窝里，手机微光中的温暖

| 类型 | 参考 | 应用方向 |
|------|------|---------|
| **App** | Day One、Journey | 日记类产品的简洁布局 |

## 三、字体规范

### 3.1 字体家族

**中文：** 思源黑体 CN (Source Han Sans CN)
**英文：** Inter
**数字：** DIN Alternate (可选)

### 3.2 字体层级系统

```
┌─────────────────────────────────────────────────┐
│  Display  48px / Light    大数字、特殊展示      │
├─────────────────────────────────────────────────┤
│  H1       32px / Light    页面主标题            │
├─────────────────────────────────────────────────┤
│  H2       24px / Medium   模块标题              │
├─────────────────────────────────────────────────┤
│  H3       18px / Medium   小标题、分组标题      │
├─────────────────────────────────────────────────┤
│  Body-L   16px / Normal   正文 (大)             │
├─────────────────────────────────────────────────┤
│  Body     16px / Normal   正文 (标准)           │
├─────────────────────────────────────────────────┤
│  Body-S   14px / Normal   正文 (小)、说明文字   │
├─────────────────────────────────────────────────┤
│  Caption  12px / Normal   辅助信息、时间戳      │
├─────────────────────────────────────────────────┤
│  Overline 10px / Medium   标签、分类标识        │
└─────────────────────────────────────────────────┘
```

### 3.3 行高规范

| 类型 | 行高倍数 | 示例 |
|------|---------|------|
| 标题 | 1.3 - 1.4 | H1: 32px × 1.3 = 41.6px |
| 正文 | 1.6 - 1.8 | Body: 16px × 1.7 = 27.2px |
| 紧凑 | 1.4 - 1.5 | 列表项、卡片内容 |
| 宽松 | 1.8 - 2.0 | 诗歌、引用、特殊内容 |

### 3.4 字重使用规范

| 字重 | 使用场景 |
|------|---------|
| Light (300) | 大标题、展示数字、装饰性文字 |
| Normal (400) | 正文、说明文字、常规内容 |
| Medium (500) | 小标题、按钮文字、强调内容 |
| Bold (700) | 重要数据、核心按钮、警示信息 |

## 四、图标风格

### 4.1 图标设计原则

**核心特征：**
- ✏️ **手绘感：** 轻微的不规则线条，避免过于机械
- 🌸 **圆润：** 圆角处理，无尖锐棱角
- 🎨 **双色：** 主色 + 浅色填充，增加层次
- 💫 **温度：** 有人情味，不是冷冰冰的几何图形

### 4.2 图标规格

| 类型 | 尺寸 | 线宽 | 风格 |
|------|------|------|------|
| 导航图标 | 24×24 px | 1.5 px | 线性 + 填充 |
| 功能图标 | 20×20 px | 1.5 px | 线性 |
| 小图标 | 16×16 px | 1 px | 线性 |
| 装饰图标 | 32×32 px | 2 px | 填充 + 渐变 |

### 4.3 图标颜色规范

| 状态 | 颜色 | 使用场景 |
|------|------|---------|
| 默认 | 中灰 #9B9590 | 未选中状态 |
| 激活 | 心语橙 #FF8A5B | 选中状态 |
| 禁用 | 浅灰 #E8E6E3 | 不可用状态 |
| 特殊 | 珊瑚粉/薄荷绿 | 点赞、成功等 |

### 4.4 图标清单 (50+ 功能图标)

**导航类 (8 个)**
- 首页、日记、匹配、聊天、我的
- 返回、关闭、更多、搜索

**功能类 (15 个)**
- 写日记、发布、编辑、删除、收藏
- 分享、复制、导出、打印、归档
- 设置、通知、帮助、反馈、关于

**社交类 (10 个)**
- 点赞、评论、私信、关注、粉丝
- 匹配、推荐、举报、屏蔽、拉黑

**状态类 (10 个)**
- 成功、失败、警告、等待、完成
- 在线、离线、忙碌、隐身、VIP

**内容类 (10 个)**
- 文字、图片、语音、视频、链接
- 表情、贴纸、主题、模板、草稿

- 🌈 低饱和配色，莫兰迪色系
- 😌 人物表情温和，动作自然

## 七、动效规范

### 7.1 动效原则

**核心理念：** 克制、流畅、有温度

**时长规范：**
- 小动效：150-200ms (按钮反馈、图标变化)
- 中动效：250-300ms (页面切换、卡片展开)
- 大动效：400-500ms (引导页、特殊转场)

**缓动曲线：**
- 进入：cubic-bezier(0.25, 0.1, 0.25, 1)
- 退出：cubic-bezier(0.25, 0.1, 0.25, 1)
- 弹性：cubic-bezier(0.68, -0.55, 0.265, 1.55)

### 7.2 常见动效

| 动效名称 | 触发场景 | 时长 | 说明 |
|---------|---------|------|------|
| 按钮按压 | 点击按钮 | 150ms | 轻微缩放 0.95 |
| 卡片展开 | 点击卡片 | 250ms | 高度展开 + 淡入 |
| 页面切换 | 导航跳转 | 300ms | 左右滑动 + 淡入淡出 |
| 加载旋转 | 数据加载 | 持续 | 360° 旋转 |
| 点赞动画 | 点击点赞 | 400ms | 放大 + 粒子散开 |
| 匹配成功 | 匹配完成 | 500ms | 两颗心靠近 + 光芒 |
| 消息到达 | 新消息 | 200ms | 从下滑入 + 弹跳 |
| 输入反馈 | 输入焦点 | 150ms | 边框颜色渐变 |

## 九、视觉检查清单

### 9.1 设计评审标准

**每次设计评审前，请确认：**

- [ ] 色彩使用符合品牌规范
- [ ] 字体层级清晰可读
- [ ] 图标风格统一一致
- [ ] 组件样式符合设计系统
- [ ] 留白充足，不拥挤
- [ ] 手绘感元素温度适中
- [ ] 动效流畅不突兀
- [ ] 深色模式适配完整
- [ ] 无障碍设计考虑周全

### 9.2 常见设计误区

❌ 色彩过于鲜艳，破坏治愈感
❌ 字体过小，影响可读性
❌ 图标过于几何，缺少温度
❌ 页面拥挤，留白不足
❌ 动效过多，干扰用户
❌ 风格不统一，拼凑感强
