### 4.2 设计标注规范

**每个页面需包含：**
- [ ] 页面名称 + 版本号
- [ ] 尺寸标注 (375×812 基准)
- [ ] 颜色标注 (Hex + Token)
- [ ] 字体标注 (字号 + 字重 + 行高)
- [ ] 间距标注 (4px 倍数)
- [ ] 交互说明 (点击、滑动、动效)
- [ ] 状态说明 (默认、Hover、Pressed、Disabled)

### 4.3 切图规范

**导出格式：**
- iOS: @2x, @3x PNG
- Android: mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi
- Web: PNG + SVG (图标)

**命名规范：**
```
模块_元素_状态@倍数.格式

示例：
btn_primary_normal@2x.png
btn_primary_pressed@2x.png
icon_home_active@2x.png
icon_home_normal@2x.png
```
