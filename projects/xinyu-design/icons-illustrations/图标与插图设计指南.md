## 一、设计风格定义

### 1.1 核心风格

**温暖治愈系手绘风格**

**关键词：** 手绘感 · 水彩质感 · 柔和配色 · 圆润造型 · 情感表达

### 1.2 视觉特征

| 特征 | 描述 | 示例 |
|------|------|------|
| **线条** | 轻微不规则，保留手绘温度 | 非完美几何 |
| **色彩** | 低饱和莫兰迪色系 | 柔和不刺眼 |
| **造型** | 圆润无尖角 | 圆角、曲线 |
| **质感** | 水彩晕染效果 | 渐变、透明度 |
| **表情** | 温和治愈 | 微笑、平静 |

## 六、图标生成工作流

### 6.1 批量生成流程

```
1. 准备提示词模板
2. 使用文生图工具批量生成
3. 统一尺寸和格式
4. 人工审核和调整
5. 导出 SVG 和 PNG
6. 整理到设计系统
```

### 6.2 质量检查清单

- [ ] 风格一致性 (线条、色彩、质感)
- [ ] 尺寸准确性 (24×24, 20×20, 16×16)
- [ ] 色彩符合规范 (主色、强调色)
- [ ] 可识别性 (缩小后仍清晰)
- [ ] 手绘感适中 (不过于机械也不过于随意)
- [ ] 圆角统一 (2px 标准)
- [ ] 导出格式完整 (SVG + PNG @2x @3x)
