# 睡了么 App 隐私政策要点

## 文档信息
- **版本号**: V1.0
- **生效日期**: 2026 年 3 月
- **更新日期**: 2026 年 3 月
- **适用范围**: 睡了么 App（iOS/Android）及其相关服务

## 一、引言

睡了么（以下简称"我们"）深知个人信息对您的重要性，并庄严承诺保护您的隐私安全。本隐私政策旨在向您说明在使用睡了么服务时，我们如何收集、使用、存储、共享和保护您的个人信息，以及您享有的权利。

**重要提示**：
- 请在使用睡了么前仔细阅读本政策
- 一旦使用睡了么服务，即表示您同意我们按本政策处理您的信息
- 如不同意，请立即停止使用并注销账号

---

## 二、信息收集

### 2.1 我们收集的信息类型

#### （一）您主动提供的信息

| 信息类型 | 收集场景 | 用途 |
|----------|----------|------|
| 手机号码 | 注册/登录 | 账号创建、身份验证、安全保护 |
| 账号密码 | 注册/登录 | 账号安全保护 |
| 个人头像 | 个人档案设置 | 个性化展示 |
| 昵称 | 个人档案设置 | 个性化展示 |
| 性别、年龄 | 健康档案（可选） | 个性化睡眠分析 |
| 身高、体重 | 健康档案（可选） | 健康建议生成 |

#### （二）我们自动收集的信息

| 信息类型 | 收集方式 | 用途 |
|----------|----------|------|
| 睡眠数据 | 手机传感器 | 睡眠监测、分析、报告 |
| 设备信息 | 系统 API | 兼容性适配、故障排查、安全风控 |
| 日志信息 | 自动记录 | 服务优化、问题分析 |
| 位置信息 | GPS/网络定位（可选） | 当地日出日落时间、时区适配 |
| 使用行为 | 埋点统计 | 产品优化、功能改进 |

#### （三）传感器使用说明

睡了么使用以下传感器提供睡眠监测服务：

- **加速度计**：检测体动，判断睡眠状态
- **陀螺仪**：辅助判断睡姿变化
- **麦克风**（可选）：检测鼾声、梦话等声音（需单独授权）
- **环境光传感器**：判断入睡/起床环境光线

**说明**：传感器数据仅在睡眠监测期间采集，不上传原始数据，仅上传分析结果。

### 2.2 征得授权同意的例外

根据相关法律法规，以下情形我们收集使用您的个人信息无需征得您的同意：

1. 与国家安全、国防安全有关的
2. 与公共安全、公共卫生、重大公共利益有关的
3. 与犯罪侦查、起诉、审判和判决执行等有关的
4. 出于维护您或其他个人的生命、财产等重大合法权益但又很难得到本人同意的
5. 所收集的个人信息是您自行向社会公众公开的
6. 从合法公开披露的信息中收集个人信息的
7. 根据您的要求签订合同所必需的
8. 用于维护睡了么服务的安全稳定运行所必需的
9. 为合法的新闻报道所必需的
10. 学术研究机构基于公共利益开展统计或学术研究所必要，且对外提供研究成果时对结果进行去标识化处理的

## 三、信息使用

### 3.1 我们如何使用您的信息

我们严格遵守法律法规规定，将收集的信息用于以下用途：

1. **提供服务**：睡眠监测、数据分析、健康建议
2. **安全保障**：账号安全、风险识别、欺诈防范
3. **产品优化**：功能改进、用户体验提升
4. **客户服务**：问题处理、投诉建议响应
5. **合法合规**：配合监管要求、司法调查

### 3.2 信息使用规则

- **最小必要原则**：仅收集提供服务所必需的信息
- **目的限定原则**：信息仅用于声明的用途
- **去标识化处理**：分析统计时使用去标识化数据
- **用户可控**：您可随时撤回授权、删除数据

## 四、信息共享与披露

### 4.1 共享原则

我们不会与睡了么以外的任何公司、组织和个人共享您的个人信息，但以下情况除外：

1. **获得您的明确同意**
2. **法定情形**：根据法律法规、诉讼争议解决需要，或按行政、司法机关依法提出的要求
3. **与授权合作伙伴共享**：仅为实现本政策中声明的目的

### 4.2 授权合作伙伴

| 合作伙伴类型 | 共享信息 | 用途 |
|--------------|----------|------|
| 云服务提供商 | 加密后的睡眠数据 | 数据存储、备份 |
| 短信服务商 | 手机号码（加密） | 验证码发送 |
| 数据分析服务商 | 去标识化使用数据 | 产品分析优化 |
| 健康平台（可选） | 睡眠数据（经您授权） | Apple Health/Google Fit 同步 |

**承诺**：所有合作伙伴均需签署严格的数据保护协议，且我们对其数据处理活动进行监督。

### 4.3 信息披露

在以下情形中，我们可能会披露您的个人信息：

1. 获得您明确同意后
2. 基于法律规定或合理依据
3. 与国家安全、国防安全有关的
4. 与公共安全、公共卫生、重大公共利益有关的
5. 与犯罪侦查、起诉、审判和判决执行等有关的

## 五、信息保护

### 5.1 技术保护措施

我们采取符合业界标准的安全防护措施保护您的个人信息：

- **传输加密**：使用 HTTPS/TLS 加密传输
- **存储加密**：云端数据采用 AES-256 加密存储
- **访问控制**：严格限制数据访问权限，最小化授权
- **安全审计**：定期进行安全审计和风险评估
- **应急响应**：建立数据安全事件应急预案

### 5.2 管理保护措施

- **人员管理**：签署保密协议，违规追责
- **流程管理**：建立数据全生命周期管理规范
- **合规认证**：通过 ISO 27001 信息安全管理体系认证
- **定期培训**：员工隐私保护意识培训

### 5.3 安全事件处理

如发生个人信息安全事件，我们将：

1. 立即启动应急预案，采取补救措施
2. 按照法律法规要求向主管部门报告
3. 通过 App 推送、短信、邮件等方式告知您
4. 配合监管部门调查处理

## 六、您的权利

### 6.1 访问权

您有权访问您的个人信息，操作路径：
- **睡眠数据**：App 首页 → 睡眠记录
- **个人信息**：我的 → 设置 → 账号与安全
- **设备信息**：我的 → 设置 → 关于我们

### 6.2 更正权

如发现信息有误，您有权要求更正：
- **自助修改**：昵称、头像等可在设置中直接修改
- **联系客服**：其他信息错误请联系客服处理

### 6.3 删除权

在以下情形中，您可以要求我们删除个人信息：

1. 我们违反法律法规或与您的约定收集使用信息
2. 我们违反规定与第三方共享您的信息
3. 您不再使用睡了么服务
4. 我们终止服务及运营

**删除方式**：我的 → 设置 → 账号与安全 → 注销账号

### 6.4 撤回同意权

您可随时撤回对特定权限的授权：

- **系统权限**：手机设置 → 睡了么 → 权限管理
- **功能授权**：我的 → 设置 → 隐私设置

**提示**：撤回部分权限可能影响相关功能使用，但不影响基础服务。

### 6.5 获取副本权

您有权获取个人信息副本：
- **操作路径**：我的 → 设置 → 数据管理 → 导出数据
- **格式支持**：PDF 报告、Excel 原始数据
- **发送方式**：加密链接发送至绑定邮箱

### 6.6 约束信息系统自动决策

如我们使用您的信息进行自动化决策（如个性化推荐），您有权要求说明并拒绝：
- **操作路径**：我的 → 设置 → 隐私设置 → 广告个性化

## 七、未成年人保护

### 7.1 保护原则

- 睡了么主要面向 18 岁以上用户
- 如您是 14 周岁以下未成年人，请在监护人指导下使用
- 我们不会故意收集未成年人个人信息

### 7.2 监护人权利

如监护人有合理理由认为我们收集了未成年人信息：
1. 请联系客服说明情况
2. 我们将核实并尽快删除相关信息
3. 监护人可要求注销未成年人账号

## 八、政策变更

### 8.1 变更情形

以下情况我们可能更新本隐私政策：

1. 服务模式变化（如新增功能）
2. 个人信息共享、转让或公开披露的主要对象变化
3. 您参与个人信息处理方面的权利及其行使方式变化
4. 我们的联络方式变化
5. 隐私政策安全风险评估表明存在高风险

### 8.2 变更通知

- **一般变更**：App 内公告、官网公示
- **重大变更**：App 推送通知、弹窗提示
- **生效时间**：通知后 7 日或您继续使用服务时

### 8.3 您的选择

如不同意变更后的政策，您有权：
1. 停止使用睡了么服务
2. 注销账号并删除个人信息

## 十、定义

- **个人信息**：以电子或其他方式记录的能够单独或与其他信息结合识别特定自然人身份或反映特定自然人活动情况的各种信息
- **个人敏感信息**：包括身份证件号码、个人生物识别信息、银行账号、通信记录和内容、财产信息、征信信息、行踪轨迹、住宿信息、健康生理信息、交易信息、14 岁以下（含）儿童的个人信息等
- **设备信息**：包括设备型号、操作系统版本、设备设置、唯一设备标识符、软硬件特征信息等
- **去标识化**：通过对个人信息的技术处理，使其在不借助额外信息的情况下，无法识别个人信息主体的过程
