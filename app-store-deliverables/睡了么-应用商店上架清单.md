## 三、视觉素材规范

### 3.1 应用图标 (App Icon)

**规格要求：**
- 尺寸：1024 x 1024 px
- 格式：PNG（无透明通道）
- 色彩模式：sRGB
- 文件大小：< 15 MB

**设计规范：**
```
设计主题：宁静、舒适、夜晚
主色调：深蓝色 (#1A1F3C) + 暖黄色 (#FFD966)
核心元素：月亮/星星 + 睡眼/梦境元素
风格：扁平化 + 轻微渐变
避免：文字、复杂细节、透明背景
```

**设计建议：**
- 使用渐变夜空背景（深蓝到紫色）
- 中央放置可爱的月亮形象（带睡眼）
- 点缀小星星增加梦幻感
- 确保在小尺寸下仍清晰可辨

### 3.2 App Store 截图 (Screenshots)

**规格要求：**
- 6.7 寸 (iPhone 14/15 Pro Max)：1290 x 2796 px
- 6.5 寸 (iPhone 11 Pro Max 等)：1242 x 2688 px
- 格式：PNG 或 JPEG
- 数量：最少 5 张，最多 10 张

**5 张截图内容规划：**

| 序号 | 主题 | 展示内容 | 文案 |
|------|------|----------|------|
| 1 | 首页/睡眠追踪 | 睡眠数据仪表盘 | "精准追踪每一晚" |
| 2 | 睡眠分析 | 睡眠周期图表 | "读懂你的睡眠" |
| 3 | 助眠音频 | 白噪音/冥想界面 | "快速入睡的秘密" |
| 4 | 智能闹钟 | 闹钟设置界面 | "在最佳时刻醒来" |
| 5 | 睡眠报告 | 周/月报告展示 | "见证睡眠改善" |

**截图设计规范：**
```
• 使用真实设备外壳框架
• 每张截图顶部添加标题文案（字体：PingFang SC Bold, 36pt）
• 文案背景：半透明渐变遮罩
• 统一色调：与应用主色调一致
• 突出核心功能区域（使用轻微高亮或指示）
• 避免状态栏干扰（可后期处理移除）
```

### 3.3 宣传视频 (App Preview)

**规格要求：**
- 时长：15-30 秒
- 分辨率：1080p (1920 x 1080) 或更高
- 格式：MP4 (H.264 编码)
- 帧率：30 fps
- 文件大小：< 500 MB
- 音频：可选（建议添加轻柔背景音乐）

**30 秒视频脚本：**

```
[0-3 秒] 开场
画面：应用图标淡入，背景为星空
文案："睡了么"

[3-8 秒] 睡眠追踪
画面：用户放置手机，切换到睡眠追踪界面
文案："自动追踪睡眠质量"

[8-15 秒] 助眠功能
画面：展示白噪音选择界面，播放轻柔音效
文案："精选助眠音频，快速入眠"

[15-22 秒] 数据分析
画面：睡眠报告图表动画展示
文案："AI 分析，个性化改善建议"

[22-27 秒] 智能闹钟
画面：闹钟界面，清晨阳光
文案："浅睡唤醒，告别起床气"

[27-30 秒] 结尾
画面：应用图标 + 下载按钮
文案："睡了么 - 每晚好眠"
```

**视频制作建议：**
- 使用设备录制 + 后期剪辑
- 添加平滑转场效果
- 背景音乐选择：轻柔钢琴/环境音
- 文案使用应用品牌字体
- 保持节奏明快，避免冗长

## 八、ASO 优化建议

### 8.1 持续优化策略

```
1. 关键词监控
   • 每周检查关键词排名

   • 关注竞品关键词策略

   • 及时回复用户评论

   • 鼓励满意用户留下好评

3. 截图 A/B 测试
   • 定期更新截图顺序
   • 测试不同文案效果
   • 根据转化率优化

4. 版本更新
   • 保持每月至少一次更新
   • 更新说明突出新功能
   • 利用更新重新激活用户
```

### 8.2 数据指标追踪

```
核心指标：
• 下载量（日/周/月）
• 转化率（浏览→下载）
• 关键词排名
• 用户评分与评论数
• 留存率（7 日/30 日）

工具推荐：
• App Store Connect 自带分析
• App Annie / data.ai
• Sensor Tower
```
