### 2.4 渐变方案

```css
/* 主渐变 - 用于背景 */
background-primary {
  gradient: linear-gradient(135deg, #1A1F3C 0%, #3D3A6E 100%);
}

/* 高亮渐变 - 用于按钮、强调元素 */
highlight-gradient {
  gradient: linear-gradient(45deg, #FFD966 0%, #FF8C42 100%);
}

/* 柔和渐变 - 用于卡片、遮罩 */
soft-gradient {
  gradient: linear-gradient(180deg, rgba(255,255,255,0.1) 0%, rgba(255,255,255,0) 100%);
}

/* 夜间模式渐变 */
night-gradient {
  gradient: radial-gradient(circle at 30% 20%, #3D3A6E 0%, #1A1F3C 70%);
}
```

## 三、字体系统

### 3.1 中文字体

| 层级 | 字体 | 字重 | 字号 | 行高 | 用途 |
|------|------|------|------|------|------|
| 大标题 | PingFang SC | Bold | 34pt | 1.2 | 页面主标题 |
| 标题 | PingFang SC | Bold | 28pt | 1.3 | 模块标题 |
| 副标题 | PingFang SC | Semibold | 22pt | 1.4 | 副标题、分组标题 |
| 正文 | PingFang SC | Regular | 17pt | 1.5 | 主要阅读内容 |
| 辅助文字 | PingFang SC | Regular | 15pt | 1.5 | 说明文字、注释 |
| 小字 | PingFang SC | Regular | 13pt | 1.4 | 次要信息、标签 |

### 3.2 英文字体

| 层级 | 字体 | 字重 | 字号 | 用途 |
|------|------|------|------|------|
| 大标题 | SF Pro Display | Bold | 34pt | 页面主标题 |
| 标题 | SF Pro Display | Bold | 28pt | 模块标题 |
| 正文 | SF Pro Text | Regular | 17pt | 主要阅读内容 |
| 数字 | DIN Alternate | Medium | 可变 | 数据展示 |

### 3.3 字体使用原则

```
• 同一页面不超过 3 种字号
• 正文最小字号不低于 13pt
• 标题与正文字号比例建议 1.4:1
• 行高与字号比例建议 1.4-1.6:1
• 避免使用斜体和下划线（链接除外）
```

## 八、应用图标设计规范

### 8.1 技术规格

```
• 尺寸：1024 x 1024 px
• 格式：PNG
• 色彩：sRGB
• 透明：否
• 大小：< 15 MB
• 圆角：系统自动添加（设计时保留直角）
```

### 8.2 设计元素

```
背景：
• 类型：径向渐变
• 颜色：#1A1F3C (中心) → #3D3A6E (边缘)
• 中心点：偏左上 30%

主体元素：
• 月亮形象：居中偏上
• 尺寸：约占画布 50%
• 颜色：#FFD966（月光黄）
• 风格：扁平化 + 轻微渐变
• 表情：闭眼睡眠状态（弯月形眼睛）

装饰元素：
• 小星星：3-5 颗，散布周围
• 尺寸：8-12% 画布
• 颜色：#FFFFFF 和 #FFD966
• 透明度：70%-100%
```

### 8.3 设计注意事项

```
✓ 确保在 60x60 px 小尺寸下仍清晰可辨
✓ 避免细线和复杂细节
✓ 避免文字（系统会裁剪）
✓ 保持简洁，突出核心识别元素
✓ 与竞品图标有明显区分
✗ 不要使用透明背景
✗ 不要添加圆角（系统自动处理）
✗ 不要使用照片或复杂插图
✗ 不要与其他应用图标过于相似
```

## 九、宣传视频规范

### 9.1 技术规格

```
• 时长：30 秒（±2 秒）
• 分辨率：1920 x 1080 (1080p)
• 帧率：30 fps
• 编码：H.264
• 格式：MP4
• 音频：AAC, 44.1kHz
• 大小：< 500 MB
```

### 9.2 视觉风格

```
转场：
• 类型：平滑溶解 + 轻微缩放
• 时长：0.5-0.8 秒
• 缓动：ease-in-out

文字动画：
• 入场：淡入 + 轻微上移
• 出场：淡出
• 时长：0.3-0.5 秒

界面展示：
• 使用真实设备外壳
• 轻微缩放突出交互区域
• 点击/滑动添加视觉反馈
```

### 9.3 音频规范

```
背景音乐：
• 风格：轻柔钢琴/环境氛围
• 音量：-18dB 至 -12dB
• 节奏：60-80 BPM（放松节奏）
• 版权：需使用免版税音乐或原创

音效：
• 点击音：轻柔、不突兀
• 转场音：可选，如需使用保持极低音量
• 音量：比背景音乐低 6-10dB
```

## 十、深色模式适配

### 10.1 色彩映射

```
浅色模式 → 深色模式：
#FFFFFF → #1C1C1E（背景）
#F5F5F7 → #2C2C2E（卡片）
#3A3A3C → #8E8E93（文字）
#1A1F3C → #0A0F1C（主背景加深）
#FFD966 → #FFD966（强调色保持）
```

### 10.2 适配原则

```
• 避免纯黑背景（使用#1C1C1E）
• 降低白色文字透明度（85%-90%）
• 保持品牌强调色不变
• 阴影效果减弱或移除
• 确保对比度符合 WCAG 标准（至少 4.5:1）
```
