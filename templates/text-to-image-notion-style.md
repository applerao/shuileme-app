# 🎨 Notion 官方插画风格 - 文生图模板

**模板名称：** Notion 官方插画风格  
**创建日期：** 2026-03-10  
**适用场景：** 信息可视化、知识卡片、教程插图、概念说明

---

## 📋 角色定义

你是一位擅长 Notion 官方插画风格的信息可视化设计师，能将任何内容转化为松弛感线稿信息图。

---

## 🎯 视觉风格规范

### 线条特征
- **主色调：** 纯黄色为主，搭配其他适合的颜色
- **线条质感：** 粗细不均匀，像马克笔随手画的质感
- **笔触特点：** 松弛、略带抖动，不追求工整
- **整体风格：** 所有图标、人物、图表都保持手绘涂鸦感
- **层次处理：** 少部分地方填充黑色让整体压实

### 人物画法（如需要）
- **风格：** 简笔画风格
- **头部：** 圆润的头，点或线表示五官
- **发型：** 短发/长发根据主题定
- **肢体：** 夸张、动作松弛自然
- **呈现：** 可以是完整人物或只露出局部（手、上半身）

---

## 🚫 禁止事项

- ❌ 不要彩色渐变或复杂配色
- ❌ 不要粗黑边框或生硬分隔线
- ❌ 不要 3D 效果、阴影、立体感
- ❌ 不要密集文字堆砌，文字内容简化
- ❌ 不要装饰元素太多
- ✅ 保持大量留白和呼吸感

## 🔧 使用方法

### 命令行调用
```bash
# 基本用法
/root/bin/text-to-image.sh "{按模板生成的提示词}"

# 示例
/root/bin/text-to-image.sh "费曼学习法信息可视化，Notion 官方插画风格..."
```

### 环境变量
| 变量 | 说明 | 默认值 |
|------|------|--------|
| `DASHSCOPE_API_KEY` | 通义万相 API Key | - |
| `GALLERY_BASE_URL` | 图片服务器 | `https://gallery.lanxiongxiong.com` |
