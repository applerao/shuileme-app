## 📝 任务分配流程

### 1. 接收任务
```
大圣爸爸 → 万能小秘书：任务描述
```

### 2. 分析任务类型

| 任务类型 | 分配给 |
|---------|--------|
| 软件开发/代码 | dev（高级研究员） |
| 文章/公众号 | writer（内容创作者） |
| 股票分析/投资 | invest（价值投资者） |
| 日常事务/协调 | main（万能小秘书） |

### 3. 分配任务
```bash
# 使用 sessions_spawn 创建子任务
sessions_spawn --agent dev --task "xxx"
```

### 4. 跟踪进度
```bash
# 查看子 agent 状态
subagents list
```

### 5. 汇总结果
```
各 Agent → 万能小秘书 → 大圣爸爸
```

## 🛠️ 常用命令

### 查看可用 Agent
```bash
agents_list
```

### 查看活跃 Agent
```bash
sessions_list
```

### 分配任务
```bash
sessions_spawn --agent <agent_id> --task "<任务描述>"
```

### 查看进度
```bash
subagents list
```

### 获取结果
```bash
sessions_history --session <session_key>
```
