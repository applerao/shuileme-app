# 配图生成指南 -《35 岁被裁员，AI 来背锅》

## 🖼️ 图 1：封面图（900x500）

### 方案 A：对比冲击型（推荐）

**AI 绘图 Prompt（Midjourney）：**
```
Split screen composition, left side: tired 35-year-old programmer 
working late at desk with messy code on screen, dark cold colors, 
exhausted expression; right side: sleek AI robot efficiently coding 
with holographic interface, tech blue and purple colors; dramatic 
divide in the middle, Chinese text "35 岁" on left and "AI" on right, 
cinematic lighting, digital art, professional illustration --ar 9:5 --v 6
```

**AI 绘图 Prompt（DALL-E 3 / 国内 AI）：**
```
左右分屏对比图。左侧：35 岁程序员深夜加班，疲惫表情，灰暗冷色调，
电脑屏幕上满是代码。右侧：AI 机器人高效工作，科技蓝紫色调，
全息界面。中间有明显分割线。左侧写"35 岁"，右侧写"AI"。
电影级灯光，数字艺术，专业插画风格。尺寸 900x500。
```

**图库搜索关键词：**
- Unsplash: "programmer tired" + "AI robot"
- Pexels: "coding late night" + "artificial intelligence"
- 组合两张图用 Canva 拼接

**自制方案（Canva/稿定）：**
1. 找一张程序员加班图（左）
2. 找一张 AI/机器人图（右）
3. 中间加一条裂痕/闪电分割效果
4. 左侧加文字"35 岁"，右侧加文字"AI"
5. 调整色调：左侧降饱和，右侧增强科技蓝

**DALL-E 3 / 国内 AI:**

**自制方案（推荐 - 用 Canva/飞书）：**
1. 打开 Canva 搜索"infographic"
2. 选择三栏布局模板
3. 每栏放一个人物图标（Canva 素材库有）
4. 添加图标：齿轮、书本、复制符号
5. 添加文字标签
6. 导出为 PNG

**优势：** 文字清晰可控，比 AI 生成更准确

### 方案 B：自制（强烈推荐）

**用 Notion/飞书文档制作：**

```
┌─────────────────────────────────────────────────────┐
│          AI 能做什么？    │    AI 做不了什么？       │
├─────────────────────────────────────────────────────┤
│  ✅ 写基础代码          │    ❌ 理解复杂业务        │
│  ✅ 生成文档            │    ❌ 做战略决策          │
│  ✅ 整理数据            │    ❌ 搞定难缠客户        │
│  ✅ 回答常见问题        │    ❌ 带团队管人          │
│  ✅ 辅助创作            │    ❌ 为结果负责          │
└─────────────────────────────────────────────────────┘
```

**步骤：**
1. 打开 Notion 或飞书文档
2. 创建一个两列表格
3. 左侧绿色背景，右侧红色背景
4. 填入上方内容
5. 截图或导出为图片
6. 用 Canva 加个边框美化

**优势：** 文字 100% 准确，排版清晰

## 🛠️ AI 绘图工具使用教程

### Midjourney（效果最佳）

**步骤：**
1. 加入 Discord：https://discord.com/invite/midjourney
2. 订阅会员：$10/月起（https://midjourney.com）
3. 进入任意 newbies 频道
4. 输入：`/imagine prompt: [上方 Prompt]`
5. 等待 1-2 分钟生成
6. 选择满意的版本，点击 U 放大
7. 下载图片

**提示：**
- 用 `--ar 16:9` 控制比例
- 用 `--v 6` 使用最新版本
- 不满意可以 `V1/V2/V3/V4` 生成变体

### 国内 AI 工具（免费/低成本）

#### 1. 文心一格（百度）
- 网址：https://yige.baidu.com
- 免费额度：每日若干次
- 输入中文 Prompt 即可

#### 2. 通义万相（阿里）
- 网址：https://wanxiang.aliyun.com
- 免费额度：新用户送积分
- 支持中文 Prompt

#### 3. 即梦 AI（字节）
- 网址：https://jimeng.jianying.com
- 免费额度：每日免费生成
- 效果好，速度快

#### 4. LiblibAI（在线 SD）
- 网址：https://liblib.art
- 免费：每日若干次

- 不满意多试几次，调整关键词

## ✅ 图片质量检查清单

生成/下载图片后检查：

- [ ] 尺寸正确（封面 900x500，插图 800x450）
- [ ] 清晰度足够（放大看不模糊）
- [ ] 文字清晰可读（如有文字）
- [ ] 色调统一（5 张图风格一致）
- [ ] 无水印（图库需选免费商用）
- [ ] 文件格式：PNG 或 JPG
- [ ] 文件大小：每张<500KB（利于加载）
