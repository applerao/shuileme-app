| 优先级 | 定义 | 通过标准 |
|--------|------|---------|

| P1 | 重要功能，严重影响体验 | ≥95% 通过 |
| P2 | 一般功能，可接受的问题 | ≥90% 通过 |
| P3 | 辅助功能，轻微问题 | ≥85% 通过 |

### 缺陷等级定义

| 等级 | 定义 | 示例 |
|------|------|------|
| P0 - 致命 | 系统崩溃、数据丢失、核心功能不可用 | App 闪退、无法登录 |
| P1 - 严重 | 主要功能异常，严重影响使用 | 打卡失败、评分错误 |
| P2 - 一般 | 功能异常但有 workaround，影响部分用户 | UI 显示问题、偶发错误 |
| P3 - 轻微 | 界面瑕疵、文案错误，不影响功能 | 错别字、颜色偏差 |
