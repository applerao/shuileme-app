## 1. 测试概述

### 1.1 测试目标
验证"睡了么"App 的核心功能正常工作，包括用户注册登录、打卡功能和性能要求。

### 1.2 测试范围
- 用户注册与登录模块
- 睡前/起床打卡功能
- 连续打卡计算与睡眠评分
- API 响应性能
- App 启动性能

### 1.3 测试类型
- 功能测试
- 表单验证测试
- 错误处理测试
- 性能测试
